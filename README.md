## ğŸ§© Features

- **Wayland** with Hyprland
- Lightweight terminal: foot
- Notifications: Mako with hyprosd-mako from https://github.com/hyprwm/contrib/tree/main/hyprosd-mako
- App launcher: fuzzel
- Terminal file manager: ranger
- Music player: cmus with theme from https://github.com/Sekki21956/cmus
- GUI tools: Nemo, Brave
- Font: FiraCode Nerd Font
- Editor: NeoVim with LazyVim
- GTK Theme: Catppuccin from https://github.com/vinceliuice/Colloid-gtk-theme
- Icon Theme: Papirus Dark with papirus-folders from https://github.com/catppuccin/papirus-folders
- Wallpaper from https://github.com/orangci/walls-catppuccin-mocha

## Installation Instructions

- Make sure your OS supports Hyprland and all the listed apps and install them
- Grab the configs and place them in ~/.config
- Enjoy!
